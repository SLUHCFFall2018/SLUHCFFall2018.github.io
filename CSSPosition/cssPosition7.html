The float property

In this example the boxes float to the left. Box 6 floats against box 5 and so would box 7 if we didn't use the clear: left; property for it.

The clear: left; property will prevent box 7 from touching any box on its left side.

The boxes after box 7 have all been floated left but resume their position after the line containing box 7.
