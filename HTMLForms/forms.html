Forms

A form control is a way to collect information from visitors to your website.

Below is a list of the form possible form controls

  • text field
  • text area
  • password field
  • radio buttons
  • checkboxes
  • drop down boxes
  • submit buttons
  • file upload fields

With forms, a user fills in information or makes a choice from some group of information and then presses a submit button.

The name of the form control along with the users input is sent to a server. The server processes the information and then takes some action.

In a form, a form tag is used along with an input tag. Each of these tags has important attributes.

The label tag is used with form controls for vision-impaired users. In the form controls below, we use the labl tag.

The label tag can be used in two different ways.

1. Wrap the label around the form control and the text that goes with it.

2. Separate the label from the form control and use the "for" attribute to indicate which form a label applies to.
