This page will illustrate some of the other features of html that we have not yet covered. Many of these features don't fit into a neatly defined category.

Every html element can have an "id" attribute. The "id" attribute uniquely identifies that element from other elements on the page. No two elements on a page can have the same "id"

An "id" attribute allows us to style an element using CSS and it allows javascript to modify an element. The "id" attribute is known as a global attribute because it can be used on any element.

Every html element can also have a "class" attribute. This is not used to uniquely identify an element but rather to indicate that an element is part of a group or "class" of elements.

The "class" attribute also allows us to style and element using CSS and it allows javascript to modify an element.

The div tag is allows you to group a set of elements together into one block-level element. Rememeber, block-level elements always have a new line inserted after them.

The div tag is an organizational element and does not make a difference in the way an element appears by itself.

Using an "id" or "class" attribute with a div tag allows you to style or position all elements within a div tag.

The span tag allows you to group a set of elements together into one inline-level element. Remember, inline-level elements do not have a new line inserted after them.

Like the div tag, the span tag is an organizational element and does not make a difference in the way an element appears by itself.

Also, like the div tag, using an "id" or "class" attribute with a span tag allows you to style or position all elements within a span tag.
